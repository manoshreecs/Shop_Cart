🔗 **Live Demo:** https://shop-cart-ecommerce-hazel.vercel.app/  
💻 **GitHub Repo:** https://github.com/manoshreecs/Shop_Cart  

---

## 📌 Overview
ShopCart is a **full-stack e-commerce web application** built using the **MERN stack**.  
It provides a seamless online shopping experience along with a powerful **seller/admin dashboard** to manage products, users, and orders.

This application is fully deployed and integrates **secure payments via Stripe** and **image management using Cloudinary**.

---

## 🚀 Tech Stack

### Frontend
- React.js (Vite)

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

### Authentication
- JWT (JSON Web Token)  

### Payments
- Stripe  

### Image Storage
- Cloudinary  

---

## ✨ Features

### 🛍️ User Features
- Browse products with images, prices, and descriptions  
- View detailed product information  
- Add/remove items from cart  
- Secure checkout using Stripe  
- View order history and status  

### 🛠️ Admin/Seller Features
- Add new products with images  
- Edit and delete products  
- Manage customer orders  
- View registered users  

### ⚙️ General Features
- JWT-based authentication  
- Fully responsive design (mobile + desktop)  
- Clean and modern UI  

---

## 🔐 Security & Middleware

- Implemented **CORS (Cross-Origin Resource Sharing)** for secure frontend-backend communication  
- Used **JWT authentication** for protected routes  
- Applied **role-based access control (User / Admin)**  
- Ensured secure API handling with validation and error management
